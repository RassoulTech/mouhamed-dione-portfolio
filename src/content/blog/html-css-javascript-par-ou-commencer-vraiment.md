Tout le développement web repose sur trois langages. Comprendre leur rôle, c'est déjà la moitié du chemin.

## HTML : la structure
C'est le **squelette** de la page : titres, paragraphes, images, liens. Si une page web était une maison, HTML serait les murs.

## CSS : le style
C'est la **décoration** : couleurs, polices, mise en page, animations. CSS rend la maison belle.

## JavaScript : le comportement
C'est l'**électricité** : ce qui rend la page interactive. Clics, formulaires, contenu dynamique.

## L'ordre d'apprentissage
1. **HTML** d'abord (quelques jours)
2. **CSS** ensuite (le plus long à maîtriser)
3. **JavaScript** en dernier (le plus puissant)

## Le piège à éviter
Sauter trop vite sur un framework (React) sans maîtriser les bases. C'est comme courir avant de marcher : tu te fatigues et tu tombes.

> Maîtrise HTML, CSS et JavaScript, et tu peux déjà construire 80 % du web. Le reste n'est que des outils par-dessus ces fondations.
