# Mouhamed Dione — Portfolio

Portfolio en ligne et CV interactif de **Mouhamed Dione**, developpeur web et gestionnaire de communication digitale base a Thies, Senegal.

> Architecte minimal · React 19 + Vite · Tailwind 3 · GSAP 3 · jsPDF · Formsubmit.

## Stack

- React 19 + Vite (rolldown)
- Tailwind CSS v3 (dark mode classe)
- GSAP 3 + ScrollTrigger pour les animations
- Lucide React pour les icones
- jsPDF (chunk lazy) pour la generation du CV PDF a la volee
- Formsubmit.co pour le formulaire de contact (zero backend)

## Scripts

```bash
npm install          # installe les deps
npm run dev          # serveur de dev (http://localhost:5173)
npm run build        # build prod dans dist/
npm run preview      # previsualise le build
```

## Deploiement

Le projet est deploye sur Vercel via un import direct du repo GitHub : chaque push sur `main` declenche un redeploy automatique.
